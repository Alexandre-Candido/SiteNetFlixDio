Estudo Digital Innovation One: Clone do site NETFLIX

Estudo realizado com base no site do NETFLIX.
Clonagem da interface do principal site de streaming mundial utilizando tecnologias simples como HTML5, CSS3 e JavaScript.
Neste projeto foi estudado:

  • Estrutura de layout
  • Técnicas de CSS3 com containers e variáveis
  • Posicionamento de elementos com Flexbox
  • Utilização de plugins Jquery

Este projeto é um "estudo" e não tem fins lucrativos.
Todas as imagens inseridas neste projeto foram retiradas dos sites IMDB e NETFLIX.
